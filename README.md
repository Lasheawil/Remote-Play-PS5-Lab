<h1>PS Remote Play Home Lab</h1>

 

<h2>Description</h2>
Project consists of installiing PS Remote Play on a Windows PC. This will allow the user to connect remotely from the Windows PC to their PS5. This requires the PS Remote Play app and a fast internet connection. It first must be enabled in the PS5's system settings along with Rest Mode settings.
<br />


<h2>Utilities Used</h2>

- <b>PS Remote Play</b> 


<h2>Environments Used </h2>

- <b>Windows 11</b> (21H2)
