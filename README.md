<h1>PS Remote Play Home Lab</h1>

 

<h2>Description</h2>
Project consists of enabling <b>Remote Play</b> on a Playstation and installing <b><a href="https://remoteplay.dl.playstation.net/remoteplay/lang/en/1100001.html?smcid=RemotePlay%3ADL_WinPC#title_wrk_bf4_1xb">PS Remote Play</b> on a Windows PC. This will allow the user to connect remotely from the Windows PC to their PS5. This requires the PS Remote Play app and a fast internet connection. It first must be enabled in the PS5's system settings along with Rest Mode settings.
<br />


<h2>Utilities Used</h2>

- <b>PS Remote Play</b> 


<h2>Environments Used </h2>

- <b>Windows 11</b> (21H2)

<h2>Program walk-through:</h2>

<p align="center">
First, set up your Playstation to enable remote play. <br/>Select "Settings" cog in upper right hand corner: <br/>
<img src="https://github.com/Lasheawil/Remote-Play-PS5-Lab/blob/main/1000004186.jpg" height="80%" width="80%"/>
<br />

<p align="center">
Select <strong>System>Remote Play</strong>, and then turn on <strong>Enable Remote Play</strong>: <br/>
<img src="https://github.com/Lasheawil/Remote-Play-PS5-Lab/blob/main/1000004187.jpg" height="80%" width="80%"/>
<br />
 <img src="https://github.com/Lasheawil/Remote-Play-PS5-Lab/blob/main/1000004185.jpg" height="80%" width="80%"/>
 <br />
 

<p align="center">
 To start your console while PS5 is in rest mode, select <b>Settings>System>Power Saving>Features Available in Rest <br>Mode:<br/></b>:
 
 <img src="https://github.com/Lasheawil/Remote-Play-PS5-Lab/blob/main/1000004184.jpg" height="80%" width="80%"/>
 <br/>

  <p align="center">
 Turn on <b>Stay Connected to the Internet</b> and <b>Enable Turning On PS5 from Network</b> <br>Then put the PS5 in rest mode.:
 <img src="https://github.com/Lasheawil/Remote-Play-PS5-Lab/blob/main/rest%20mode.jpg" height="80%" width="80%"/>
 <br/>
   
<p align="center">
Next, on a PC, download <b><a href="https://remoteplay.dl.playstation.net/remoteplay/lang/en/1100001.html?smcid=RemotePlay%3ADL_WinPC#title_wrk_bf4_1xb">PS Remote Play</b> and save it to your desired location. Run the installation file and follow the on screen instructions to install.<br/>
 <br><img src="https://github.com/Lasheawil/Remote-Play-PS5-Lab/blob/main/remote%20play%20install%20screen.png" height="80%" width="80%"/>
<br/>

<p align="center">
Once the installation is complete, select <b>Launch PS Remote Play</b> and click <b>Finish</b><br/>
<img src="https://github.com/Lasheawil/Remote-Play-PS5-Lab/blob/main/launch%20remote%20finished.png" height="80%" width="80%"/>
<br />

<p align="center">
The next screen will require you to sign into your PS account<br/>
<img src="https://github.com/Lasheawil/Remote-Play-PS5-Lab/blob/main/sign%20in.png" height="80%" width="80%"/>
<br />

<p align="center">
On the next screen, select the system you want to connect to<br/>
<img src="https://github.com/Lasheawil/Remote-Play-PS5-Lab/blob/main/select%20system.png" height="80%" width="80%"/>
<br />

 
<p align="center">
When the connection is completed, the PS5 homescreen will open on your desktop. See the taskbar at the bottom of the screen.<br/>
<img src="https://github.com/Lasheawil/Remote-Play-PS5-Lab/blob/main/completed%20.png" height="80%" width="80%"/>
